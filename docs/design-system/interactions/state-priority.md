# STATE PRIORITY

When multiple states exist simultaneously:

1. disabled
2. loading
3. error
4. focus
5. hover
6. active
7. default

Higher-priority states override lower-priority states.

Examples:

- disabled suppresses hover
- error overrides focus color
- loading disables interaction
