# AI EXECUTION RULES

Before creating new components:

1. search for reusable primitives
2. reuse existing patterns
3. prefer composition over duplication
4. preserve token consistency
5. avoid introducing unnecessary variants
6. preserve naming conventions
7. prefer semantic tokens over raw values

AI-generated output must integrate into the existing system instead of creating parallel patterns.

When conflicts exist:

1. accessibility wins
2. usability wins
3. consistency wins
4. visual aesthetics come last
